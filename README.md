# FlyMan2

「フライマン2」

（概要・テーマ）
この作品は、HAL時代に作成したフライマンをUnrealEngine5でリメイクしたゲームです。
UnrealEngine5とC++を勉強するために開発した作品になります。

(開発時期・期間)
DICO株式会社時の2023年6月～2023年9月(4ヶ月)

（開発ツール）
・ゲームエンジン：UnrealEngine5(5.1.1)
・言語：C++, ブループリント
・開発環境：Visual Studio2022

（チームメンバー）
プログラマー1人（自分）

（ゲームシステム）
ジャンプしながらゴールを目指すゲームです。
ゴールに着くとゲームクリアーになります。
落ちるとゲームオーバーになります。

(操作方法)
・AとDキーでプレイヤーキャラクターを左右移動させることができます。
・Spaceキーでプレイヤーキャラクターをジャンプさせることができます。

(起動方法)
「FlyMan2(ビルド)」というフォルダーを開いてください。
その中にある「FlyMan2」というアプリケーションをダブルクリックしてください。
